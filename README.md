# ClawLabs

> AI-Powered Portfolio of Apps, SaaS & Tools

## Overview

ClawLabs is an AI-first portfolio platform showcasing a curated collection of apps, SaaS products, and developer tools — all built with Claude Code and AI-assisted development workflows. Designed for developers, indie hackers, and AI enthusiasts, ClawLabs demonstrates what's possible when you combine modern full-stack engineering with the speed and intelligence of Anthropic Claude.

Built on a robust stack including **Next.js**, **TypeScript**, **React**, and **Tailwind CSS**, ClawLabs delivers a fast, responsive experience while **Prisma** and **NeonDB** power the data layer. Authentication is handled seamlessly via **NextAuth**, and GitHub integration through **Octokit** keeps open-source projects connected and discoverable.

## Key Features

- **Claude Code-Driven Development** — Every project in the portfolio was co-piloted by Anthropic Claude, showcasing real-world AI-assisted software development from ideation to deployment.
- **Full-Stack Project Showcase** — Explore web apps, REST APIs, dashboards, and automation tools built with Next.js, TypeScript, and React.
- **Open Source Transparency** — Most projects are publicly available on GitHub, giving developers direct access to codebases and architecture decisions.
- **Framer Motion & Radix UI** — Polished, accessible interfaces with smooth animations that make browsing the portfolio a premium experience.
- **Vercel-Deployed & Production-Ready** — Every tool ships fast and scales reliably on Vercel's edge infrastructure.

## How It Works

ClawLabs uses a Next.js App Router architecture with server components and API routes backed by Prisma ORM and a serverless NeonDB Postgres database. Projects are catalogued with metadata, tags, and GitHub stats fetched via Octokit. NextAuth manages secure session handling, while Tailwind CSS and Radix UI ensure a consistent, accessible design system throughout.

## FAQ

### What is ClawLabs and what kind of projects does it showcase?
ClawLabs is an AI-first portfolio platform featuring apps, SaaS products, and developer tools built with Claude Code and AI-assisted development. Projects span full-stack web apps, APIs, dashboards, and automation tools — all built using Next.js, TypeScript, and React.

### What does it mean to build software with Claude Code?
Building with Claude Code means using Anthropic's Claude AI as a co-pilot throughout the entire development process — from architecture planning and code generation to debugging and iteration. ClawLabs uses this AI-assisted workflow to ship production-ready projects faster and with greater consistency.

### Is ClawLabs open source and can I see the code on GitHub?
Yes — most projects showcased on ClawLabs are open source and publicly available on GitHub. The platform integrates with GitHub via Octokit to surface live repository data, making it easy to explore codebases, star projects, or fork them for your own use.

### What tech stack is used to build ClawLabs projects?
ClawLabs projects are built with a modern full-stack TypeScript stack including Next.js, React, Prisma ORM, NeonDB (serverless Postgres), NextAuth, Tailwind CSS, Radix UI, and Framer Motion, with deployments managed on Vercel.

### Can indie hackers or solo developers use ClawLabs as inspiration for AI-assisted projects?
Absolutely. ClawLabs is specifically designed for developers and indie hackers curious about AI-first development workflows. Each project demonstrates how a solo developer can leverage Claude Code to rapidly design, build, and ship full-stack SaaS products and tools.

### How does ClawLabs handle authentication and data storage?
ClawLabs uses NextAuth for secure, flexible authentication and Prisma ORM connected to a NeonDB serverless Postgres database for scalable data storage. This stack enables fast iteration while keeping the platform production-ready and secure.
